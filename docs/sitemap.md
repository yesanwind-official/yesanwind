## 1. 사이트맵 개요

```
예산윈드오케스트라 웹사이트
│
├── 홈 (/)
│
├── 오케스트라 소개 (/about)
│   ├── 인사말 (/about/greeting)
│   ├── 연혁 (/about/history)
│   ├── 조직도 (/about/organization)
│   └── 오시는 길 (/about/location)
│
├── 단원 소개 (/members)
│   ├── 지휘자 (/members/conductor)
│   └── 단원 (/members/players)
│       ├── 목관 파트 (/members/players?section=woodwind)
│       ├── 금관 파트 (/members/players?section=brass)
│       └── 타악 파트 (/members/players?section=percussion)
│
├── 공연 안내 (/concerts)
│   ├── 정기연주회 (/concerts/regular)
│   ├── 기획공연 (/concerts/special)
│   └── 지난 공연 (/concerts/archive)
│
├── 갤러리 (/gallery)
│   ├── 사진 (/gallery/photos)
│   └── 영상 (/gallery/videos)
│
├── 게시판 (/board)
│   ├── 공지사항 (/board/notice)
│   └── 언론보도 (/board/press)
│
└── 관리자 (/admin) [비공개]
    ├── 대시보드 (/admin)
    ├── 공연 관리 (/admin/concerts)
    ├── 단원 관리 (/admin/members)
    ├── 갤러리 관리 (/admin/gallery)
    └── 게시판 관리 (/admin/board)
```

## 4. URL 설계 원칙

### 4.1 URL 규칙

| 규칙 | 예시 |
|------|------|
| 소문자 사용 | `/concerts`, `/gallery` |
| 하이픈(-) 구분 | 해당 없음 (단일 단어 사용) |
| 복수형 사용 (목록) | `/members`, `/concerts` |
| ID 기반 상세 | `/concerts/[id]` |
| 쿼리 파라미터 (필터) | `/members/players?section=brass` |

### 4.2 동적 라우트

| 패턴 | 용도 |
|------|------|
| `/concerts/[id]` | 공연 상세 페이지 |
| `/members/players/[id]` | 단원 상세 페이지 |
| `/gallery/photos/[id]` | 사진 앨범 상세 |
| `/gallery/videos/[id]` | 영상 상세 |
| `/board/notice/[id]` | 공지사항 상세 |
| `/board/press/[id]` | 언론보도 상세 |
